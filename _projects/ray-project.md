description: |
  Using the principles of statistical mechanics and thermodynamics, we can define eight different ensembles. In addition to the Guggenheim ensemble, (the "fifth" statistical ensemble), one can modified the thermodynamic constraints to define three additional statistical ensembles. Brown, Hill, and Ray developed the isoenthalpic-isobaric ensemble (N, P, H), the grand-isochoric adiabatic (µ, V, L) ensemble, and the grand-isobaric adiabatic (µ, P, R) ensemble, using the Legendre-Laplace mapping procedure. Together with the microcanonical ensemble, they form a set of four adiabatic ensembles, in which the value of a heat function is constant, rather than the temperature.
